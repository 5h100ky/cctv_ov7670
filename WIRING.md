# OV7670 → RP2040 Zero 배선표

## 전원 주의사항
- OV7670 은 **3.3V 전용** (5V 연결 금지 → 카메라 손상)
- XCLK 는 RP2040 Zero GP15 에서 PWM 으로 자동 생성됨

## 핀 연결

| OV7670 핀 | RP2040 Zero GPIO | 비고 |
|-----------|-----------------|------|
| 3V3 / VCC | 3V3             | 카메라 전원 |
| GND       | GND             | 공통 GND |
| SIOD      | GP0             | SCCB 데이터 (I2C SDA) |
| SIOC      | GP1             | SCCB 클럭 (I2C SCL)  |
| VSYNC     | GP2             | 프레임 동기 신호 |
| D0        | GP4             | 데이터 비트 0 (LSB) |
| D1        | GP5             | 데이터 비트 1 |
| D2        | GP6             | 데이터 비트 2 |
| D3        | GP7             | 데이터 비트 3 |
| D4        | GP8             | 데이터 비트 4 |
| D5        | GP9             | 데이터 비트 5 |
| D6        | GP10            | 데이터 비트 6 |
| D7        | GP11            | 데이터 비트 7 (MSB) |
| PCLK      | GP12            | 픽셀 클럭 |
| HREF      | GP13            | 수평 라인 유효 신호 |
| XCLK      | GP15            | 카메라 마스터 클럭 입력 (~8 MHz) |
| RESET     | 3V3             | 리셋 비활성화 (HIGH 고정) |
| PWDN      | GND             | 파워다운 비활성화 (LOW 고정) |

> GP3, GP14 는 미사용 (여유 핀)

## 풀업 저항
- SIOD, SIOC 에 **4.7 kΩ** 풀업 저항을 3.3V 와 연결

## 모듈 종류별 참고
- **OV7670 (렌즈 포함, FIFO 없는 기본형)**: 위 배선 그대로 사용
- **OV7670 + AL422 FIFO 버전**: FIFO 핀 무시하고 직접 병렬 출력 핀 연결
- **OV7675**: 핀 호환, 동일 배선 사용 가능
